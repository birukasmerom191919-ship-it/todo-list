# To-Do List App

A simple and responsive To-Do List application built using HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Save tasks using Local Storage
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage
